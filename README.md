## 💎 Design Philosophy: "Void & Incandescence"

This portfolio is an exercise in **Majestic Minimalism**. It rejects generic UI patterns in favor of a custom design system that prioritizes deep negative space and high-fidelity technical interaction.

-   **Cinematic Glassmorphism**: High-fidelity surfaces with real-time blur and custom lighting orchestrations.
-   **Orbital Tech-Stack**: A dynamic, GPU-accelerated constellation visualization driven by golden-ratio spirals.
-   **Liquid Background Energy**: Multi-layered ambient glows that morph in real-time, responding to user intent.
-   **HUD Interaction Model**: High-density, technical information displays (Skills/Principles) optimized for rapid scanning.

## 🛠 Engineering Deep Dive

### 🏗️ Architectural Core
The system utilizes a **Decoupled Content-Driven Engine**. The UI is a pure presentation layer, completely agnostic of the data it renders, which is managed via a strict YAML protocol.

-   **Type-Safe Orchestration**: 100% TypeScript coverage with strict mode enabled.
-   **Schema Enforcement**: Data integrity is guaranteed via [Zod](https://zod.dev/) validation layers.
-   **GPU-Accelerated Motion**: Complex orchestrations powered by [Framer Motion](https://www.framer.com/motion/) with `transform-gpu` optimizations.
-   **Modern Layout Engine**: Built with **Tailwind CSS 4**, utilizing advanced container queries and fluid typography.

### ⚡ Performance Optimization
-   **Atomic CSS**: Minimal bundle size through utility-first styling.
-   **Static Export Strategy**: Fully optimized for edge deployment (GitHub Pages/Vercel).
-   **Asset Orchestration**: Specialized Next.js Image handling for LCP (Largest Contentful Paint) excellence.
-   **Hydration Efficiency**: Strategic use of `"use client"` directives to minimize main-thread blocking.

## 🚀 Deployment & Configuration

### Content Orchestration
The entire portfolio recalibrates based on `content/portfolio.yml`. No code changes are required for content updates.

```yaml
personalInfo:
  name: "Shesher Hasan"
  title: "Full-Stack Engineer"
  tagline: "Building high-performance systems and scalable web applications"
```

### Installation Protocol

1. **Clone & Initialize:**
   ```bash
   git clone https://github.com/myth-tools/portfolio.git
   npm install
   ```

2. **System Verification:**
   ```bash
   npm run check   # Runs Biome linting & TypeScript validation
   ```

3. **Development Cycle:**
   ```bash
   npm run dev     # Starts the high-fidelity dev server
   ```
